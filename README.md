# 📝 Task Manager Web App

A simple and efficient Task Manager that allows users to create, manage, and track daily tasks.

Built to practice full-stack development and CRUD operations.

---

## 🚀 Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Clean and responsive UI

---
## 🛠️ Tech Stack

- **Frontend:** Next.js, React.js  
- **Styling:** CSS / Tailwind CSS (if applicable)  
- **Deployment:** Vercel / Netlify  
- **Version Control:** Git & GitHub

## ⚡ Installation

Clone the repository

```bash

cd task-manager
npm install
npm run dev
```
